# BMplan.md 멀티 버티컬 비전 업데이트 계획

**작성일:** 2026-03-26
**상태:** 확정 대기
**대상 파일:** `docs/BMplan.md` (현재 1604줄, v1.0)

## Context

강상모 대표의 궁극적 비전: ValueConnect X는 IT/Tech 분야를 시작점으로, **대한민국 모든 분야의 미슐랭 3스타급 최고급 인재**들이 모이는 네트워크로 발전한다. 요리사(미슐랭 스타 셰프), 재즈 뮤지션, 도자기 공예가, 예술가, 연예인, 가수 등 각 분야 최정상 인재를 포괄하는 방향.

현재 BMplan.md는 IT/Tech 시니어(엔지니어, PM, VP, CTO)와 기업 CEO 연결에 100% 초점이 맞춰져 있으며, 다른 분야 확장에 대한 언급이 전혀 없음.

## Work Objectives

1. 기존 IT/Tech 중심 BM을 훼손하지 않으면서, 궁극적 비전으로서의 멀티 버티컬 확장 방향을 BMplan.md에 통합
2. 각 분야별 확장이 현실적으로 어떤 단계를 거치는지 로드맵 제시
3. 수익 모델이 분야별로 어떻게 변형/확장되는지 구조화

## Guardrails

### Must Have
- 기존 섹션 1~7의 핵심 수치/구조는 그대로 유지
- IT/Tech가 "Phase 1 (현재)" 임을 명확히 하여 기존 내용과 자연스럽게 연결
- 각 버티컬(분야)의 특성에 맞는 차별화된 수익 모델 언급
- 비전 선언문이 Executive Summary와 결론 양쪽에 반영

### Must NOT Have
- 기존 재무 전망/Unit Economics 수치 변경
- 비현실적인 타임라인 (예: 1년 안에 10개 분야 진출)
- 기존 섹션 순서 변경 또는 대규모 구조 개편

## 5. 멀티 버티컬 확장 비전: "대한민국의 미슐랭 3스타 네트워크"

### 5.1 비전 선언
- 궁극적 목표: 각 분야 최정상 인재들의 교차 네트워크
- "미슐랭 3스타"의 의미: 해당 분야에서 동료들이 인정하는 Top 1% 전문가
- IT/Tech는 시작점이지 종착점이 아님

### 5.2 버티컬 확장 로드맵

#### Phase 1 (현재~Year 2): IT/Tech 기반 확립
- 현재 BM 그대로 (섹션 1~4 참조)
- Core 200명, 기업 20곳 달성이 확장의 전제 조건
- 이 단계에서 "초대 기반 검증 시스템"의 프로토콜 완성

#### Phase 2 (Year 2~3): 인접 전문직 확장
- **대상 분야:** 디자인/크리에이티브 (UX/UI, 산업디자인, 브랜드), 경영/금융 전문가 (CFO, 투자심사역, 회계사)
- **확장 논리:** IT/Tech CEO들이 이미 필요로 하는 인접 분야 → 기존 수요 기반 자연스러운 확장
- **수익 모델:** 기존 25% 수수료 구조 동일 적용 가능
- **버티컬별 Core 멤버:** 각 20~30명 시작

#### Phase 3 (Year 3~5): 창의/문화 분야 진출
- **대상 분야:**
  - 요리 (미슐랭 스타 셰프, 소믈리에, 파티시에)
  - 음악 (재즈 뮤지션, 클래식 연주자, 프로듀서)
  - 공예/예술 (도자기 공예가, 현대미술 작가, 갤러리스트)
  - 엔터테인먼트 (배우, 가수, 감독, 작가)
- **확장 논리:** IT/Tech 네트워크의 CEO/리더들이 문화 분야 인재와의 교류 니즈 보유 (기업 브랜딩, 행사, 개인적 관심)
- **수익 모델 변형:** (아래 5.3에서 상세)

#### Phase 4 (Year 5+): 범분야 통합 네트워크
- **대상 분야:** 의료 (전문의, 한의학), 법률 (로펌 파트너), 스포츠 (감독, 트레이너), 학술 (교수, 연구자)
- **궁극적 모습:** "한 분야의 3스타가 다른 분야의 3스타를 만나는 유일한 공간"
- 크로스 버티컬 매칭의 가치 극대화

### 5.3 분야별 수익 모델 설계

#### 5.3.1 전문직 분야 (디자인, 경영, 법률, 의료)
- 기존 채용 수수료 25% 모델 거의 동일 적용
- 프로젝트 매칭 (4.6과 연동): 컨설팅/자문 프로젝트 중개 수수료 15%
- 분야별 Market Intelligence 리포트 판매

#### 5.3.2 창의/문화 분야 (요리, 음악, 공예, 예술)
- **에이전시형 매칭 수수료:** 셰프 초빙, 아티스트 콜라보레이션, 공연/전시 기획 → 프로젝트 총 비용의 15~20%
- **브랜드 콜라보레이션 중개:** 기업 × 아티스트 협업 프로젝트 매칭 → 중개 수수료 10~15%
- **프리미엄 이벤트:** 미슐랭 셰프의 VCX 멤버 전용 프라이빗 디너, 재즈 뮤지션의 소규모 프라이빗 공연 → 참석료 + 스폰십
- **마스터 클래스:** 3스타급 전문가의 소수정예 워크숍 (도자기, 요리 등) → 참가비 + 온라인 콘텐츠 구독

#### 5.3.3 엔터테인먼트 분야
- **매니지먼트 연계:** 배우/가수의 기업 행사 출연 중개 → 출연료의 10~15%
- **IP 콜라보레이션:** 기업 × 연예인 브랜드 협업 → 프로젝트 수수료
- **프라이빗 네트워킹:** CEO × 연예인 교류 이벤트 → 프리미엄 참석료

### 5.4 크로스 버티컬 시너지

#### 5.4.1 교차 네트워킹의 가치
- IT CEO가 미슐랭 셰프를 만나 사내 다이닝 프로그램 기획
- 갤러리스트가 테크 기업의 오피스 아트 큐레이션 수주
- 재즈 뮤지션이 기업 브랜딩 이벤트의 프라이빗 공연 수주
- 도자기 공예가가 기업 VIP 선물 커스텀 제작

#### 5.4.2 플라이휠 효과
- 다양한 분야의 3스타 인재 → 네트워크의 희소성 증가
- 희소성 → 기존 멤버의 자부심과 충성도 강화
- 충성도 → 더 적극적인 추천과 참여
- 참여 → 더 풍부한 이벤트와 콘텐츠 → 새로운 분야의 3스타 유입

### 5.5 버티컬 확장 시 검증 시스템

#### 5.5.1 분야별 검증 기준 원칙
- IT/Tech: 동료 평판 + 경력 + CEO 기준 검증 (현재 방식)
- 요리: 미슐랭/아시아 50 Best 수상 이력, 동료 셰프 추천
- 음악: 주요 페스티벌 출연, 앨범 크레딧, 동료 뮤지션 추천
- 예술: 갤러리 전시 이력, 아트 페어 참가, 큐레이터 추천
- 엔터테인먼트: 주요 작품 출연, 수상 이력, 업계 평판

#### 5.5.2 공통 원칙
- "동료가 인정하는 3스타" — 모든 분야에서 Peer Review 기반 검증
- 추천인 책임 시스템은 분야 불문 동일 적용
- 각 버티컬에 분야 전문 큐레이터(Advisory Board) 구성

### Step 4: 경쟁 우위/해자 섹션에 멀티 버티컬 관점 추가

**위치:** 기존 섹션 6 (새 번호 7) - 경쟁 우위와 해자

**작업 내용:**
- 6.1 네트워크 효과에 "크로스 버티컬 네트워크 효과" 하위 항목 추가
  - 단일 분야 네트워크는 모방 가능하지만, 분야 간 교차 네트워크는 극도로 모방 어려움
  - "IT CEO + 미슐랭 셰프 + 재즈 뮤지션"이 한 공간에 있는 네트워크는 세계적으로도 유례 없음
- 6.3 신뢰 기반 진입 장벽에 "분야별 Peer Review 검증"이 해자를 더 강화한다는 내용 추가

**수락 기준:**
- 크로스 버티컬 네트워크 효과가 경쟁 우위의 하나로 명시됨
- 기존 해자 내용을 보완하는 형태 (대체가 아닌 추가)

### Step 5: 결론 섹션 비전 업데이트

**위치:** 결론 (마지막 섹션)

**작업 내용:**
- 기존 결론의 "핵심 성공 요인" 5개 유지
- 6번째 성공 요인 추가: "멀티 버티컬 확장: IT/Tech를 넘어 모든 분야의 3스타 인재가 모이는 궁극적 비전"
- 마지막 볼드 문단을 확장하여 멀티 버티컬 비전 포함
  - 기존: "이 사업의 해자는 코드가 아니라 관계입니다..."
  - 추가: "그리고 이 관계는 IT/Tech에서 멈추지 않습니다. 미슐랭 셰프, 재즈 뮤지션, 도자기 공예가, 현대미술 작가 — 각 분야의 3스타가 서로를 발견하고 연결되는 공간. 이것이 ValueConnect X의 궁극적 목표입니다."

**수락 기준:**
- 결론에 멀티 버티컬 비전이 자연스럽게 통합됨
- 기존 결론의 톤과 메시지가 유지되면서 확장됨
- 재무 목표 수치는 변경 없음 (멀티 버티컬 수익은 Year 5+ 이므로 3년 전망에 미포함)

## Success Criteria

1. BMplan.md가 v2.1로 업데이트되고, 멀티 버티컬 비전이 Executive Summary, 신규 섹션 5, 경쟁 우위, 결론 총 4곳에 반영됨
2. 기존 IT/Tech 중심 BM (섹션 1~4)의 수치와 내용이 변경 없이 유지됨
3. 새 섹션 5가 비전 선언, 4단계 로드맵, 분야별 수익 모델, 크로스 시너지, 검증 시스템, Year 5 수익 시나리오를 모두 포함
4. 섹션 번호 체계가 일관성 있게 정리됨
5. `npm run build` 영향 없음 (문서 파일만 변경)
