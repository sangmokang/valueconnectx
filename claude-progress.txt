[결정 1] 프레임워크: Next.js 14 App Router + TypeScript strict

[결정 3] 데이터 패칭: Server Component SSR + SWR (클라이언트 인터랙션)
  이유: 주간 데이터 특성상 공격적 캐싱 가능 (revalidate: 3600)

[결정 4] Materialized View 우선 활용
