# ValueConnect X 비즈니스 모델 계획서

**작성일:** 2026년 3월
**버전:** 2.1 (PRD v5.1, v4.2.1 + ValueHire Intelligence 시너지 반영 + 멀티 버티컬 확장 비전)

## Executive Summary

ValueConnect X는 검증된 핵심 인재와 기업 의사결정자(CEO/Founder/C-Level)를 신뢰 기반으로 연결하는 **Private Talent Network**입니다. 현재 채용 시장의 '선택적 채용 × 선택적 인재' 현상에 응답하며, 관계 기반의 프리미엄 헤드헌팅 서비스를 제공합니다.

**v2.1 궁극적 비전 — 멀티 버티컬 확장:**
ValueConnect X의 궁극적 목표는 IT/Tech를 넘어 **대한민국 모든 분야의 미슐랭 3스타급 최고급 인재**가 모이는 Private Talent Network입니다. 요리(미슐랭 스타 셰프), 음악(재즈 뮤지션), 공예(도자기 공예가), 예술(현대미술 작가), 엔터테인먼트(배우, 가수) 등 각 분야 최정상 인재를 단계적으로 포괄하며, IT/Tech는 이 여정의 시작점입니다. (상세: 섹션 5 참조)

**v2.0 핵심 업데이트 — ValueHire Intelligence 시너지:**
ValueConnect는 **ValueHire(valuehire.cc)** 서비스를 통해 채용 시장의 수요·공급 데이터를 실시간으로 누적하고 있습니다. 이 **Hire Intelligence 자산**을 활용하여:
1. **B2C 시장 인사이트 구독** (커피값 한 잔, 월 9,900원) → 개인 유저에게 정제된 시장 정보 제공 → **안정적 현금 흐름 확보**
2. **CEO 커피챗 Ignition** → "이런 인재가 관심 있습니다" 방식으로 CEO를 자연스럽게 태핑 → **채용 퍼널 점화**
3. **데이터 인사이트 → 인재 리텐션** → 최고의 인재가 시장 정보를 위해 플랫폼에 정착 → **네트워크 자산 강화**

**핵심 비즈니스 모델:**
- 채용 성사 시 25% 수수료 수취 (주력 매출)
- 셀프 소개 시 이직자 500만원 보상 + VC 수익 80%
- 지인 추천 시 이직자 200만원 + 추천인 30% 수수료 + VC 수익 62%
- CEO 커피챗 Ignition을 통한 자연스러운 채용 퍼널 점화
- **[NEW] ValueHire B2C 구독 (월 9,900원)** → 초기 현금 흐름 + 인재 풀 확대

**3년 재정 전망:**
- Phase 2 (6개월): 월 성사 1건, MRR 2,000만원 + B2C 구독 MRR 99만원~

## 1. 현재 비즈니스 모델

### 1.1 Revenue Model 상세

#### 1.1.1 수수료 구조: 25%

일반 서치펌 수수료(20~25%)와 동등하지만, ValueConnect X는 **검증된 핵심 인재만 연결**하므로 그 가치를 정당화합니다.

| 항목 | 설명 |
|------|------|
| **수수료율** | 연봉의 25% (이직자 연봉 기준) |
| **계산 기준** | 이직자가 실제 받게 될 예상 연봉 |
| **지급 시기** | 채용 성사 확인 후 14일 이내 |
| **발생 조건** | 셀프 소개, 지인 추천, CEO 커피챗 모두 적용 |

**수수료 산정 예시:**

| 이직자 연봉 | 수수료 |
|------------|--------|
| 8,000만원 | 2,000만원 |
| 1억원 | 2,500만원 |
| 1.5억원 | 3,750만원 |
| 2억원 | 5,000만원 |

#### 1.1.2 Case 1: 셀프 소개 (Self Introduction)

멤버가 본인을 직접 소개하여 채용이 성사되는 경우입니다.

**구조:**

```
기업 ← (25% 수수료) → ValueConnect X
                    ↓
              이직자 500만원 고정
              + VC 나머지 (약 2,000만원, 연봉 1억 기준)
```

**배분:**

| 항목 | 연봉 1억 기준 |
|------|-------------|
| **수수료** | 2,500만원 |
| **이직자 보상** | 500만원 고정 |
| **VC 순수익** | 2,000만원 |
| **VC 마진율** | 80% |

**설계 논리:**
- 셀프 소개는 멤버의 능동적 행동 → 인센티브 최대화
- 고정 500만원은 대기업 사내 추천 보상금과 동등 이상 수준
- 업계에서 유일한 후보자 리워드 구조

#### 1.1.3 Case 2: 지인 추천 (Referral)

Core 멤버가 다른 인재를 추천하여 채용이 성사되는 경우입니다.

**구조:**

```
기업 ← (25% 수수료) → ValueConnect X
                    ├→ 이직자 200만원 고정
                    ├→ 추천인 (수수료의 30%)
                    └→ VC 나머지
```

**배분:**

| 항목 | 연봉 8천만 | 연봉 1억 | 연봉 1.5억 | 연봉 2억 |
|------|-----------|---------|-----------|---------|
| **수수료** | 2,000만원 | 2,500만원 | 3,750만원 | 5,000만원 |
| **이직자 보상** | 200만원 | 200만원 | 200만원 | 200만원 |
| **추천인 (30%)** | 600만원 | 750만원 | 1,125만원 | 1,500만원 |
| **VC 순수익** | 1,200만원 | 1,550만원 | 2,425만원 | 3,300만원 |
| **VC 마진율** | 60% | 62% | 약 65% | 66% |

**설계 논리 (v4.2.1 개선):**
- 이직자 보상을 500만원→200만원으로 조정한 이유:
  - 지인 추천은 '더 좋은 직장 기회 + 연봉 인상'이라는 본질적 보상이 이미 존재
  - Case 1과의 인센티브 역전 방지 (셀프소개 유인 보존)
- 추천인 비율을 20%→30%로 상향한 이유:
  - 추천인은 실질적으로 후보자 발굴·설득·이직 결심 유도를 수행하는 헤드헌터 역할
  - 이 기여에 상응하는 정당한 보상 필요
  - 기업 입장: 대신 헤드헌팅 비용 일부를 추천인에게 분배하는 개념

#### 1.1.4 Case 3: CEO 커피챗 Ignition → 채용 성사

CEO에게 막연하게 "커피챗 해보세요"가 아니라, **"이러이러한 인재가 관심이 있는데 커피챗 해보시겠어요?"** 라고 구체적으로 연결하는 것이 핵심입니다.

**Ignition 구조 (v2.0 업데이트):**

```
[ValueHire Intelligence]
  시장 데이터 분석 → 특정 인재의 이직 관심 시그널 감지
  ↓
[ValueConnect 태핑]
  링크드인에서 해당 기업 CEO를 찾아 접촉
  "귀사에 관심 있는 이런 경력의 인재가 있습니다. 커피 한 잔 하시겠어요?"
  ↓
[CEO 수락]
  구체적 인재 정보가 있으니 수락율 높음
  ↓
[1:1 커피챗 진행]
  자연스러운 만남 → 서로에 대한 이해
  ↓
[Ignition 발생]
  CEO: "이런 사람이 더 있나요?" → VCX 기업 클라이언트 전환
  인재: "이런 기회가 더 있나요?" → VCX Core 멤버 정착
  ↓
[채용 성사 (Case 1/2 동등 수수료 적용)]
```

**v1.0 대비 핵심 변화:**

| 항목 | v1.0 (기존) | v2.0 (Ignition) |
|------|-----------|----------------|
| **CEO 접근 방식** | "커피챗 세션 공개해주세요" (막연) | "이런 인재가 관심 있는데 만나보시겠어요?" (구체적) |
| **인재 접근 방식** | 멤버가 세션에 자발적 신청 | ValueHire 데이터로 관심 시그널 감지 → 매칭 |
| **전환율** | CEO 세션 공개 의존 (수동적) | VCX가 능동적 매칭 → 전환율 2~3배 향상 |
| **스케일** | CEO 1명당 1세션 | CEO 1명당 다수 인재 매칭 가능 |
| **차별점** | CEO가 먼저 손 내밈 | **양방향 Ignition** — CEO도 인재도 자연스럽게 점화 |

**왜 Ignition이 작동하는가:**
1. **CEO 입장**: "막연한 네트워킹" 아닌, "내가 원하는 인재를 실제로 만나는 것" → 시간 ROI 극대화
2. **인재 입장**: "내가 관심 있는 회사의 CEO가 나를 만나고 싶어 한다" → 자존감 + 기회
3. **VCX 입장**: 양측 모두 자연스럽게 VCX 생태계로 진입 → BM 가속화

**수수료:** Case 1/2와 동일한 25% 적용
**신뢰:** 약식 헤드헌팅 계약 동의 (VCX 우회 채용 방지)

#### 1.1.5 VC 마진 하한선 원칙 (v4.2.1 신규)

수수료의 **50% 이상**은 반드시 VC가 확보하는 구조입니다.

**이유:**

- 지속 가능한 서비스 운영 필수 조건

**검증:**

| 케이스 | 최저 연봉 기준 | 최저 마진율 | 하한선 충족 |
|--------|-------------|-----------|-----------|
| Case 1 (셀프 소개) | 8천만원 | 75% | ✓ |
| Case 2 (지인 추천) | 8천만원 | 60% | ✓ |

```

  → CEO 반응: "오, 만나보고 싶다!" (높은 전환율)
```

#### Ignition 프로세스 상세

```
Step 1: [ValueHire 데이터 분석]
  ├─ 인재의 이력서/프로필 분석
  ├─ 이직 관심 시그널 감지 (구독 활동, 시장가치 조회 빈도 등)

  ├─ 해당 기업 CEO를 링크드인에서 식별
  ├─ 개인화된 메시지: "귀사에 관심 있는 이런 인재가 있습니다"
  └─ 구체적 인재 정보(익명화)로 호기심 자극

Step 3: [커피챗 성사]
  ├─ CEO 수락 → 양측 일정 조율
  ├─ 1:1 커피챗 진행 (자연스러운 대화)
  └─ 서로에 대한 이해 + 신뢰 구축

### 1-A.4 데이터 인사이트 → 인재 리텐션 전략

#### 핵심 가설

> 최고의 인재에게 지속적으로 가치 있는 데이터 인사이트를 제공하면, 그들이 플랫폼에 정착하여 이탈하지 않는다.

#### 인재가 계속 돌아오게 만드는 4가지 인사이트

| 인사이트 | 제공 주기 | 인재의 반응 | 정착 효과 |
|---------|---------|-----------|---------|
| **"지금 내 시장가치는?"** | 분기 1회 | "내 연봉이 시장 대비 어디쯤인지 알 수 있다" | 분기마다 재방문 |
| **"내 직군 채용 트렌드"** | 주간 | "이 기술이 뜨고 있구나, 준비해야겠다" | 매주 확인 습관 |
| **"이직 타이밍 시그널"** | 실시간 | "지금이 움직일 때인가?" → 의사결정 지원 | 중요 시점에 의존 |
| **"맞춤 CEO 커피챗 추천"** | 월 1~2건 | "나에게 맞는 CEO가 만나고 싶어 한다" | 프리미엄 경험 |

#### 리텐션 퍼널

```
[1단계] 무료 가입 → 시장 트렌드 맛보기
  ↓ (30% 전환)
[2단계] Plus 구독 → 상세 인사이트 + 시장가치 리포트
  ↓ (인사이트 가치 체감)
[3단계] VCX Core 멤버 전환 → CEO 커피챗 + 채용 기회
  ↓ (커리어 임팩트 경험)
[4단계] 장기 정착 → 추천 활동 + 커뮤니티 기여
  ↓
[결과] 인재 네트워크 자산 → 기업에게 더 높은 가치 → 수수료 수익 극대화
```

- 마지막까지 진정성 있는 기업만 만나고 싶다

**VCX 솔루션:**
- **비공개 포지션 큐레이션:** opt-in 기반, 월 최대 2건만 발송
  - 멤버가 원할 때만 수신 가능
  - 언제든 구독 해지 가능
- **CEO Coffee Chat 비밀 신청:** CEO 수락 후에만 신원 공개
  - 초기 신청 시 익명 유지
  - 심사 단계에서 프라이버시 보호
- **Opt-Out 원클릭:** 포지션 거절 시 이유 묻지 않음
- **커뮤니티-채용 분리 원칙:** 조찬, 익명 게시판 활동이 채용 데이터로 활용되지 않음

**경제 효과:**
- 현직 유지 → 안정성 있는 이직 기회 평가
- 진정성 있는 기업만 매칭 → 성사율 향상
- 멤버 신뢰 → 장기 관계 유지 → 반복 거래

**HR 진입 흐름:**

```
CEO: "우리 CHO/CHRO는 VCX에서 참여해도 될까?"
  ↓
ValueConnect: 1:1 면담 (신뢰 확인)

  ↓

  ↓
조건 합의 및 입사
  ↓

## 5. 멀티 버티컬 확장 비전: "대한민국의 미슐랭 3스타 네트워크"

> **궁극적 비전:** ValueConnect X는 IT/Tech 시니어 인재 네트워크에서 출발하여, 대한민국 모든 분야의 **미슐랭 3스타급 최정상 인재**들이 서로를 발견하고 연결되는 유일한 공간으로 발전합니다.

### 5.1 비전 선언

#### 5.1.1 "미슐랭 3스타"의 의미

**미슐랭 3스타**는 단순한 비유가 아닙니다. 각 분야에서 **동료들이 인정하는 Top 1% 전문가**, 즉 "그 분야에서 그 사람을 만나기 위해 일부러 찾아갈 가치가 있는 사람"을 의미합니다.

| 분야 | "3스타"의 기준 | 예시 |
|------|-------------|------|
| **IT/Tech** | 동료 엔지니어/PM이 인정하는 시니어, CEO급 의사결정자 | CTO, VP Engineering, Staff Engineer |
| **요리** | 동료 셰프가 인정하는 장인, 미슐랭/아시아 50 Best 수준 | 미슐랭 스타 셰프, 소믈리에, 파티시에 |
| **음악** | 동료 뮤지션이 인정하는 연주자/프로듀서 | 재즈 뮤지션, 클래식 연주자, 음악 프로듀서 |
| **공예/예술** | 큐레이터·동료 작가가 인정하는 창작자 | 도자기 공예가, 현대미술 작가, 갤러리스트 |
| **엔터테인먼트** | 업계가 인정하는 크리에이터·퍼포머 | 배우, 가수, 감독, 작가 |

#### 5.1.2 왜 멀티 버티컬인가

**핵심 가설:** 각 분야의 최정상 인재들은 **같은 수준의 다른 분야 전문가**를 만나고 싶어 합니다. IT CEO가 미슐랭 셰프와 대화하고, 재즈 뮤지션이 현대미술 작가와 교류하는 — 이런 **크로스 버티컬 네트워킹**은 현재 한국에 존재하지 않습니다.

**시장 공백:**
- 각 분야별 폐쇄 네트워크는 존재 (요리사 모임, 뮤지션 커뮤니티 등)
- 그러나 **분야를 가로지르는 3스타급 인재 네트워크**는 전무
- "한 분야의 3스타가 다른 분야의 3스타를 만나는 유일한 공간" — 이것이 ValueConnect X의 궁극적 포지셔닝

#### 5.1.3 IT/Tech는 시작점이지 종착점이 아니다

현재 BMplan 섹션 1~4에 기술된 IT/Tech 중심 BM은 이 비전의 **Phase 1**입니다. IT/Tech에서 검증된 초대 기반 검증 시스템, 커뮤니티-채용 분리 원칙, 신뢰 기반 네트워크 운영 프로토콜이 다른 분야 확장의 기반이 됩니다.

### 5.4 크로스 버티컬 시너지

#### 5.4.1 교차 네트워킹의 구체적 가치

**시나리오 예시:**

| 시나리오 | 연결 | 경제적 가치 |
|---------|------|-----------|
| IT CEO가 사내 다이닝 프로그램 기획 | IT CEO × 미슐랭 셰프 | 셰프 초빙 계약 연 3,000만원 |
| 갤러리스트가 테크 기업 오피스 아트 큐레이션 수주 | 갤러리스트 × IT 스타트업 | 큐레이션 프로젝트 2,000만원 |
| 재즈 뮤지션이 기업 브랜딩 이벤트 프라이빗 공연 | 재즈 뮤지션 × 기업 | 공연료 + 연간 계약 1,500만원 |
| 도자기 공예가가 기업 VIP 선물 커스텀 제작 | 도자기 공예가 × 기업 | 연간 납품 계약 1,000만원 |
| 엔터 크리에이터가 IT 스타트업 브랜드 앰배서더 | 가수/배우 × 스타트업 | 앰배서더 계약 5,000만원 |

#### 5.4.2 플라이휠 효과

```
다양한 분야의 3스타 인재 유입
  ↓
네트워크의 희소성 증가 ("여기서만 만날 수 있는 사람들")
  ↓
기존 멤버의 자부심과 충성도 강화
  ↓
더 적극적인 추천과 참여
  ↓
더 풍부한 이벤트와 콘텐츠
  ↓
새로운 분야의 3스타 유입 (시작 지점으로)
```

**핵심 인사이트:** 단일 분야 네트워크(IT만, 요리만)는 동종 업계 커뮤니티와 차별화가 어렵습니다. 그러나 **"IT CEO + 미슐랭 셰프 + 재즈 뮤지션 + 현대미술 작가"가 같은 조찬 테이블에 앉는 경험** — 이것은 세계적으로도 유례가 없으며, ValueConnect X만이 제공할 수 있는 가치입니다.

### 5.5 버티컬 확장 시 검증 시스템

#### 5.5.1 공통 원칙: Peer Review 기반 검증

모든 분야에서 동일하게 적용되는 **3대 검증 원칙**:

| 원칙 | 설명 | 적용 |
|------|------|------|
| **동료 인정** | "동료가 인정하는 3스타" — 해당 분야 전문가 2인 이상의 추천 | 모든 분야 동일 |
| **추천인 책임** | 나쁜 멤버 → 추천인에게도 영향 (기존 시스템 동일) | 모든 분야 동일 |
| **Advisory Board** | 각 버티컬에 분야 전문 큐레이터 3~5인 구성 | 분야별 별도 구성 |

#### 5.5.2 분야별 검증 기준

| 분야 | 정량 기준 | 정성 기준 | 검증 주체 |
|------|---------|---------|----------|
| **IT/Tech** | 경력 10년+, VP/Director급 이상 | 동료 평판, CEO 기준 검증 | 기존 Core 멤버 + VCX 운영팀 |
| **요리/F&B** | 미슐랭/아시아 50 Best, 주요 대회 수상 | 동료 셰프 추천, 레스토랑 평판 | F&B Advisory Board |
| **음악/공연** | 주요 페스티벌 출연, 앨범 크레딧 10건+ | 동료 뮤지션 추천, 공연 실적 | 음악 Advisory Board |
| **공예/예술** | 주요 갤러리 전시 3회+, 아트 페어 참가 | 큐레이터 추천, 작품 포트폴리오 | 예술 Advisory Board |
| **엔터테인먼트** | 주요 작품 출연 5건+, 수상 이력 | 업계 평판, 소속사 확인 | 엔터 Advisory Board |

#### 5.5.3 검증 프로세스 (IT/Tech 이외 분야)

```
Step 1: 기존 멤버 또는 Advisory Board 추천
  ↓
Step 2: 정량 기준 확인 (수상, 경력, 작품 이력)
  ↓
Step 3: 동료 2인 이상의 Peer Review (전화/대면)
  ↓
Step 4: Advisory Board 최종 승인
  ↓
Step 5: VCX 운영팀 면담 (커뮤니티 적합성 확인)
  ↓
Core 멤버 등록
```

| B2C 구독 비중 (Year 3) | 0% | 10.9% | 안정적 MRR |

**누적 손익 곡선:**
- Year 1: 1.76억원 (**B2C 구독 덕분에 1년차부터 흑자 가능**)
- Year 2 말: 17.66억원 (누적 흑자 가속)
- Year 3 말: 44.92억원 (본격 수익화 + B2C MRR 안정)

**핵심 인사이트:** B2C 구독은 절대 금액보다 **현금 흐름의 예측 가능성**이 중요합니다. 채용 수수료는 건당 2,500만원이지만 변동성이 크고, B2C 구독은 매월 안정적으로 들어오는 MRR로 운영비를 커버하며 **생존 확률을 높이는 역할**을 합니다.

## 7. 경쟁 우위와 해자(Moat)

### 7.1 네트워크 효과

#### 7.1.1 동적 네트워크 성장

**Supply Side (인재):**
- Core 20명 → 50명 → 100명 → 200명 (기하급수)
- 각 Core는 평균 2~3명의 Intro 추천 가능
- 추천 효과로 인한 유기적 확장
- **[v2.0] B2C 구독자 10,000명+ → 데이터 인사이트로 인재가 자발적으로 정착**

**Demand Side (기업):**
- 첫 채용 성사 → 재거래율 60%
- 기업 수: 1→3→10→20곳 (1년마다 3배)
- 반복 거래로 인한 LTV 증대
- **[v2.0] CEO Ignition → "이런 인재가 관심 있다"는 메시지에 CEO가 자발적 유입**

**Data Flywheel (v2.0 신규):**
- ValueHire 데이터 누적 → B2C 인사이트 품질 향상 → 인재 정착 → CEO Ignition 성공률 상승 → 채용 성사 데이터 재누적
- **이 플라이휠은 시간이 갈수록 가속되며, 경쟁사가 따라오기 어려운 데이터 해자를 형성**

**결과:** 멤버 100명 = 기업 접근 기회 500개 (1명당 5건)

#### 7.1.2 멤버-기업 간 Virtuous Cycle

```
더 좋은 멤버 유입
  ↓
기업의 채용 성공률 ↑
  ↓
더 많은 기업 유입
  ↓
더 좋은 기회 제공
  ↓
더 좋은 멤버 유입 (시작 지점으로)
```

#### 7.1.3 크로스 버티컬 네트워크 효과 (v2.1 신규)

**단일 분야 네트워크는 모방 가능합니다.** IT 인재 커뮤니티, 셰프 모임, 뮤지션 네트워크는 각각 존재합니다. 그러나 **분야를 가로지르는 3스타급 인재 네트워크**는 극도로 모방이 어렵습니다.

**왜 모방 불가능한가:**

| 요소 | 단일 분야 | 크로스 버티컬 |
|------|---------|-------------|
| **진입 장벽** | 해당 분야 전문성만 필요 | 다수 분야의 검증 역량 + Advisory Board |
| **네트워크 효과** | 선형적 (n명 → n개 연결) | 지수적 (n분야 × m명 → n×m개 연결) |
| **대체재** | 업계 모임, 동문회 등 다수 | 전무 — VCX만 가능 |
| **록인 효과** | 낮음 (다른 커뮤니티로 이동 용이) | 높음 (다분야 접점을 제공하는 곳이 없음) |

**"IT CEO + 미슐랭 셰프 + 재즈 뮤지션"이 같은 테이블에 앉는 네트워크** — 이것은 경쟁사가 10년을 투자해도 쉽게 복제할 수 없는 VCX만의 해자입니다.

### 7.2 데이터 자산

#### 7.2.1 채용 데이터의 가치

**VCX가 보유한 데이터:**
- 500건 이상의 채용 기록 (Year 3)
- 직급별, 산업별, 기술별 연봉 데이터
- CEO/VP 레벨의 오퍼 조건 (equity, signing bonus)
- 이직자의 재직율, 승진율, 만족도

**경쟁사가 얻기 어려운 이유:**
- 비공개 포지션만 거래 (공개 JD 없음)
- CEO와의 직접 계약 (HR 중개 최소화)
- 관계 기반 → 투명한 정보 공유
- 감정적 신뢰 → 솔직한 피드백

#### 7.2.2 데이터 활용

| 활용처 | 가치 |
|--------|------|
| **HR/CHRO** | 연봉 기준선 설정, 포지션 큐레이션 |
| **VC** | 시장 분석, 포트폴리오 회사 벤치마킹 |
| **AI 매칭** | LLM 학습 데이터 (매치율 정밀도) |
| **학술 기관** | 한국 IT 시장 트렌드 분석 |

### 7.3 신뢰 기반 진입 장벽

#### 7.3.1 멤버 검증 시스템

**초기 20명의 강력한 검증:**
- 직접 커피챗 + 업계 평판 확인
- CEO 수준의 확인 프로세스
- 추천인 책임 시스템 (나쁜 멤버 → 추천인도 영향)

**결과:** 누구나 들어올 수 없는 네트워크 → 입장권의 가치

#### 7.3.2 커뮤니티-채용 분리 원칙

**핵심 신뢰 기반:**
- 멤버는 조찬에서 솔직하게 이야기 가능
- 조찬 발언이 채용 데이터로 활용되지 않음
- 익명 게시판 = 진짜 고민을 나눌 수 있는 공간
- 이 신뢰가 다른 플랫폼을 모방 불가능하게 함

#### 7.3.3 분야별 Peer Review 검증이 해자를 강화 (v2.1 신규)

멀티 버티컬 확장 시, 각 분야의 **전문 Advisory Board**가 Peer Review를 수행합니다(섹션 5.5 참조). 이는 단순한 이력서 확인이 아니라 **"동료가 인정하는 3스타"** 기준의 질적 검증입니다.

이 검증 시스템은 분야가 추가될수록 **복제 비용이 기하급수적으로 증가**합니다:
- 1개 분야: Advisory Board 1개 + 검증 프로토콜 1세트
- 7개 분야: Advisory Board 7개 + 분야별 맞춤 검증 프로토콜 7세트
- 경쟁사가 이를 따라하려면 각 분야의 3스타급 인재를 먼저 확보해야 하지만, 그 인재들은 이미 VCX에 있습니다.

### 8.3 기술 리스크
