# ValueConnect X
## 운영 계획서 (Operation Plan)
### v5.1 · 2026년 3월

---

## Executive Summary

ValueConnect X는 초기 30명의 Core Member 확보부터 월 3건 채용 성사까지 3단계 10개월 로드맵으로 운영됩니다. 이 계획서는 조직 구성, 멤버 확보 및 관리, 기업 유치, 커뮤니티 운영, 일일/주간/월간 루틴, 리스크 관리, Phase별 전환을 상세히 정의합니다.

---

## 1. 운영 조직 구성

### 1.1 초기 팀 구성 (Phase 1, 1~2개월)

| 직책 | 담당 업무 | 역할 설명 |
|------|---------|---------|
| **CEO / Founder** | 전략, 기업 유치, 주요 거래 | 초기 30명 Core Member 모집, CEO/Founder 네트워크 활용 |
| **Head of Operations** | Member 확보, Community 운영, 일일 관리 | Member 온보딩, 이탈 관리, Community Board 검토 |
| **Lead Engineer** | 시스템 개발, 기술 운영 | Phase 1 기능 구현 (Invite, Login, Profile, Directory, Position Board) |
| **Product Manager** | Feature 기획, UX 검증 | Member/기업 피드백 수렴, 기능 우선순위 결정 |

**팀 구성 원칙:**
- 초기 4명으로 시작 (Founder 포함)
- 모두가 Operations에 참여 (개발자도 Member 온보딩 지원)
- Member 신뢰 확보가 최우선

### 1.2 Phase별 필요 인력 증가

| Phase | 기간 | 주요 추가 인력 | 조직 규모 |
|-------|------|--------------|---------|
| **Phase 1** | 1~2개월 | - | 4명 (Founder, Ops, Engineer, PM) |
| **Phase 2** | 3~5개월 | Head of Partnerships (기업 관계), 커뮤니티 매니저 | 6명 |
| **Phase 3** | 6~10개월 | Data Analyst, 추천 시스템 Engineer, HR Consultant | 8명 |

**Phase 2 추가 인력:**
- **Head of Partnerships**: 기업 파이프라인 관리, CEO Coffee Chat Session 큐레이션, 채용 성사 지원
- **Community Manager**: Community Board 모니터링, 멤버 engagement 활성화

**Phase 3 추가 인력:**
- **Data Analyst**: 추천 시스템 성능 분석, Member 매칭 효율성 모니터링
- **Recommendation System Engineer**: AI 매칭 알고리즘 개발 및 고도화
- **HR Consultant (Part-time)**: 채용 분쟁 상담, Member 직무 매칭 정확성 검증

## 2. Member 확보 및 관리 운영

### 2.1 Core Member 30명 초기 확보 전략

#### 2.1.1 발굴 경로 (3가지 채널)

**Channel 1: Founder 개인 네트워크 (목표: 15명)**

```
대상: Founder가 직접 알고 있는 핵심인재 (전직 동료, 투자자, CEO 지인)

프로세스:
1) Founder가 1:1로 전화/커피챗 (20분)
   - 서비스 철학 설명 (관계 기반 인재 네트워크)
   - 개인적 신뢰 기반 초대

2) 초대 이메일 발송 (Personalized)
   - 왜 이 사람인지 명확히
   - 네트워크의 기대 가치 설명
   - 초대 링크 (24시간 유효)

3) 온보딩 (30분 Video Call)
   - Member Profile 작성 가이드
   - LinkedIn 검증
   - 네트워크 비전 공유

일정: Phase 1 주차 1~3 (3주)
성공 지표: 90% 이상 가입 완료
```

**Channel 2: 기업 임원진 추천 (목표: 10명)**

```
대상: 기업 CEO/COO가 추천하는 핵심 부하직원 또는 이전 동료

프로세스:
1) CEO/COO 1:1 미팅 (30분)
   - "핵심 인재 3~4명 추천해줄 수 있나요?" 형식
   - 추천 기준 설명 (조직 영향력, 동료 평판 등)

2) 추천인 정보 수집
   - 이름, 직무, 회사, 연락처
   - 간단한 추천 사유

3) Founder가 직접 초대
   - "OO CEO가 강력 추천했습니다" 메시지
   - 개인적 신뢰 기반 느낌

일정: Phase 1 주차 2~4 (3주)
성공 지표: 80% 이상 가입 완료
```

**Channel 3: Member 추천 (목표: 5명)**

```
대상: 초기 가입 Member가 추천하는 지인

프로세스:
1) 초기 10~15명 Member 온보딩 후
   Founder가 개인 메시지로 요청

2) "네트워크를 함께할 5명을 추천해주세요"
   - 조직에서 실질적 영향력 있는 사람
   - 각자 왜 추천하는지 이유 포함

3) 추천 받은 사람에게 초대
   - "OO가 당신을 추천했습니다"
   - 개인적 신뢰의 연쇄 구조

일정: Phase 1 주차 3~4 (2주)
성공 지표: 70% 이상 가입 완료
```

#### 2.1.2 선발 기준 (객관적 검증)

Core Member는 PRD 정의 기준 **3개 이상 충족**해야 합니다.

| 기준 | 판단 방법 | 검증 도구 |
|------|---------|---------|
| **조직 영향력** | 해당 인재가 떠나면 조직에 직접적 영향 | LinkedIn 경력, 추천인 피드백 |
| **동료 평판** | 업계에서 반복적으로 추천되는 인물 | 소개자 의견, 업계 평판 |
| **실행력** | 실제 성과/제품/프로젝트 단독 완수 경험 | LinkedIn, Portfolio, 인터뷰 |
| **기술 깊이** | 특정 영역의 전문성 (최소 5년 이상) | 이력서, 프로젝트 경력 |
| **성장 궤도** | 리더십 포지션으로의 가능성 있음 | 직급 변화, 책임 범위 확대 |
| **커리어 일관성** | 전문 분야 집중 (Job-hopper 아님) | LinkedIn 경력, 설명 |

**검증 프로세스:**
1) 온보딩 시 LinkedIn URL 필수 제출
2) Head of Operations가 프로필 검증 (1건당 5분)
3) 3개 이상 기준 충족 확인 후 수락
4) 부적합 판정 시: 정중히 대기 안내 후 향후 기회 제안

#### 2.1.3 초대 메시지 템플릿

#### 2.2.2 Endorsed Member 추천 및 선발 프로세스

**Step 1: 추천 요청 (Phase 2 주차 5)**

```
타겟: 가입 후 커피챗 최소 2회 이상 참여한 Core Member

메시지:
"ValueConnect를 함께할 핵심인재를 추천해주세요.
귀하가 추천하는 분은 당신을 1차 신뢰 보증인으로 참여하게 됩니다."

추천 양식:
- 이름, 직무, 회사, 연락처
- 추천 사유 (2~3줄)
- 해당인이 모르는 추천인가, 아니면 사전 동의했나

기대: Core Member당 평균 2~3명 추천
→ 30명 × 2.5 = 75명 추천 풀
→ 선발율 40% 기대 (최종 30명)
```

**Step 2: ValueConnect 검증**

```
기준: Core Member의 추천 기반 + 수용성 판정

검증 항목:
1) 추천인과의 관계 검증
   - 실제 직무 관계자인가
   - 근거 있는 추천인가

2) 기본 자격 심사
   - LinkedIn 프로필 확인 (최소 5년 경력)
   - 현재 조직에서의 실질적 역할

3) 네트워크 적합성
   - 기존 Member와의 중복도 낮은가
   - 커뮤니티 특성에 맞는가

통과 기준: 위 3개 항목 모두 충족
```

**Step 3: 초대 및 온보딩**

```
초대 메시지:
"ValueConnect의 [추천인]님이 당신을 추천했습니다."

온보딩:
- Core Member와 동일한 프로세스
- 추천인이 1차 신뢰 역할임을 명시
- 커뮤니티 가이드라인 상세 설명

멤버 표시: 프로필에 "Endorsed by [추천인]" 표시
```

### 2.3 Member 온보딩 프로세스

#### 2.3.1 전체 Flow

```
Step 1. 초대 이메일 발송 (즉시)
   ↓
Step 2. 이메일 클릭 및 인증 (24시간 유효)
   ↓
Step 3. 계정 생성 (이메일/비밀번호 또는 Google/LinkedIn SSO)
   ↓
Step 4. Member Profile 작성 (필수)
   ↓
Step 5. LinkedIn 연동 (필수)
   ↓
Step 6. 약관 동의 (Privacy Policy, Community Guidelines)
   ↓
Step 7. 네트워크 참여 (Onboarding 완료)
   ↓
Step 8. 첫 Engagement (커피챗 신청 또는 자기소개 작성)
```

#### 2.3.2 Member Profile 작성 가이드

| 항목 | 요구사항 | 검증 |
|------|---------|------|
| **Name** | 실명 필수 | 자동 (필수 필드) |
| **Current Company** | 현재 회사명 | LinkedIn과 일치 확인 |
| **Title** | 현재 직무 | 자동 |
| **Professional Field** | 최대 3개 태그 (전문 분야) | 사전 정의된 태그 리스트 |
| **Years of Experience** | 총 경력 (연수) | 자동 계산 (LinkedIn 기반) |
| **Bio** | 자기소개 (최대 200자) | 자동 제출 |
| **LinkedIn URL** | 프로필 링크 | 필수 + 실시간 검증 |

**작성 시간: 평균 10분**

| **정보 유출** | 멤버 정보를 외부에 공유 / 스크래핑 시도 | 중복 프로필, 비정상 접근 로그 | 즉시 계정 정지 + 법적 대응 |
| **악성 행동** | 다른 멤버 괴롭힘, 사기 시도 | 신고 3건 이상 | 경고 후 제명 |
| **스팸/마케팅** | 커뮤니티에 상업 광고 또는 스팸 | 게시물 리뷰, 신고 | 게시물 삭제 + 경고 + 2차 위반 시 제명 |

| **비활성** | 90일 연속 활동 없음 | 로그인/참여 기록 부재 | 3회 안내 후 자동 휴면 계정 처리 |

#### 2.5.2 신고 및 조사 프로세스

```
Step 1. 신고 (Community Member)
   → "신고" 버튼으로 게시물/멤버 신고
   → 신고 사유 입력 (5가지 카테고리)

Step 2. 자동 분류 (System)
   → 신고 유형에 따라 우선순위 결정
   → Severity High: 즉시 검토
   → Severity Medium: 24시간 내 검토

Step 3. 수동 검토 (Head of Operations)
   → 신고 내용 사실 확인 (피해자 인터뷰 포함)
   → 커뮤니티 가이드라인 위반 여부 판정

Step 4. 대응 (3가지 경로)
   경로 A: 위반 확인 안 됨 → 신고자/피신고자 개별 설명
   경로 B: 위반 확인, 경고 수준 → 1차 경고 + 가이드 재교육
   경로 C: 위반 확인, 심각 수준 → 계정 제명 + 법적 검토
```

예시:

- 유효기간: 입사일로부터 6개월 이내 퇴직 시 50% 환불
```

**정산 절차:**

```
Step 1: 입사 확정 (Offer Letter 발행 후)
   → VC에 입사 통지
   → 입사일, 연봉 기록

Step 2: 입사 확인 (입사 후 3영업일)
   → Member 또는 기업으로부터 입사 확인

Step 3: 수수료 청구 (입사 후 7일)
   → 기업에 인보이스 발행
   → 결제 기한: 30일

Step 4: 수수료 분배 (결제 후)
   → VC 수수료: 85~90% 보유
   → Member Self Introduction Reward (해당 시): 10~15% 분배
   → 기록 및 정산
```

**기술적 방어:**

```

- 1분 내 10 프로필 조회 → 경고 (자동 이메일)

- 비정상 탐색 패턴 → IP 차단 (수동 검토 후)

탐지 알고리즘:
- API 요청 패턴 분석 (Supabase Logs)
- 비정상 세션 플래깅 (Mixpanel)
- 지리적 이상 탐지 (IP 지역 변화)
```

```

| **급여 분쟁** | Member와 기업 간 연봉 협상 문제 | 중재 (계약서는 둘 사이) |
| **입사 후 30일 이내 퇴직** | Member가 채용 후 빠르게 이탈 | 환불 조건 (6개월 이내 50% 환불) 적용 |
| **정보 악용** | 기업이 CEO Coffee Chat에서 만난 인재를 다른 방식으로 접근 | Head Hunting Agreement 위반 처리 |

#### 6.3.2 분쟁 해결 절차

```
Step 1. 신고 및 초기 검토 (24시간)
   - 피해자 (Member 또는 기업)의 신고 접수
   - Head of Operations가 사실 관계 정리
   - Head Hunting Agreement 검토

Step 2. 당사자 개별 인터뷰 (48시간)
   - Member 및 기업 담당자와 전화 인터뷰
   - 문서 증거 수집 (이메일, 메시지 등)

Step 3. 법무 검토 (72시간)
   - 변호사와 협의
   - Head Hunting Agreement 위반 여부 판정
   - 배상액 또는 처벌 수준 판단

Step 4. 합의 또는 강제 집행
   합의 경로:
   - VC에서 중재안 제시
   - 양당사자 동의 시 합의금 지급

   강제 경로:
   - 법적 절차 진행
   - 배상금 청구 또는 손해배상 소송
```

#### 6.3.3 Head Hunting Agreement 강제 사례

```
기업이 CEO Coffee Chat을 통해 만난 인재를
개별 채용하려고 시도한 경우:

Step 1. VC가 기업에 통지
   "Head Hunting Agreement에 따라,
   이 채용도 ValueConnect 수수료 대상입니다."

Step 2. 기업이 거부 시
   → 법무팀에서 공식 법적 통지 발송
   → 수수료 청구서 발행

Step 3. 미정산 시
   → 소액 채권 추심 진행
   → 최악의 경우 소송 절차 진행
```

### 6.4 Member 도덕적 해이 대응

#### 6.4.1 위험 행동 및 감지

| 행동 | 위험도 | 감지 방법 | 대응 |
|------|--------|---------|------|
| **다중 계정** | 높음 | 이메일/IP 중복 | 계정 통합 또는 삭제 |
| **부정 추천** | 중간 | 추천인과의 관계 검증 불일치 | 경고 + 추천권 일시정지 |
| **개인정보 거짓** | 높음 | LinkedIn 검증 불일치 | 경고 → 계정 정지 |

#### 7.1.2 Phase 2 추가 기능 및 운영

**개발 추가 기능:**
- Coffee Chat Flow 자동화
- CEO Coffee Chat Session 관리
- Community Board 전체 오픈
- AI 기반 Member 매칭 (초기)

**조직 변화:**
- Head of Partnerships 채용 (기업 관계)
- Community Manager 채용 (게시판 운영)
- Operations 역할 확대 (채용 성사 지원)

**운영 방식 변화:**

```
Phase 1: Founder-driven
- CEO가 직접 Member 20명 초대
- 기업 CEO에게 직접 연락
- 모든 채용을 CEO가 추적

Phase 2: Team-driven
- Head of Partnerships가 기업 파이프라인 관리
- 자동화된 Re-engagement 캠페인
- 채용 성사 기록 및 정산 자동화
```

### 7.2 Phase 2→3 전환 (5개월 말, 약 7월 초)

#### 7.2.1 성공 기준

#### 7.2.2 Phase 3 전략 변화

**추천 시스템 도입:**

```
성과 지표:
- Member 장기 체류율 (12개월 이상)

- Revenue per company (평균 수수료)

- 월간 분석 → 인사이트 도출
- 기업 피드백 수집 → 포지션 품질 개선
- Member 피드백 수집 → 네트워크 설계 개선
```

| **채용 추적** | 100% Manual | 100% Manual | 80% Auto (정산 포함) |
| **Community 모더레이션** | 50% Manual | 80% Manual | 70% Auto (필터) |
