## 1. Executive Summary

### 1.1 핵심 명제

모든 조직에는 일을 실제로 처리하는 핵심인재가 존재한다. 그 인재들은 조직에서 희생을 감수하면서도 일을 끝까지 완수하고, 삶과 일이 일치된 사람들이다. 조직 내 대다수가, 타 부서가, C레벨이 이미 그 존재를 알고 있다. 보석과 같은 사람들이다.

이 사람들을 찾아서 연결하고, 평생의 커리어 파트너가 되는 것이 ValueConnect X의 목표다. 커뮤니티도, 플랫폼도, AI 매칭도 — 전부 이 관계를 만들고 유지하기 위한 도구일 뿐이다.

### 1.2 서비스 철학: 장(場)을 이해하는 사람들의 플랫폼

Pierre Bourdieu는 사회적 삶을 장(field)으로 설명했다. 각 장에는 고유한 규칙이 있고, 그 규칙을 내면화한 사람만이 진정한 플레이어가 된다.

커리어라는 장 역시 마찬가지다. 스펙은 입장권이지만, **자본(capital)을 가진 사람들과의 연결**이 실제 게임을 결정한다. ValueConnect X는 이 사회적 자본이 실질적인 기회로 전환되는 구조를 설계한다.

**우리의 테제:** "네트워크의 질은 구성원의 질을 초과할 수 없다. 하지만 올바른 구조가 주어진다면, 개인의 자본이 집단의 자본으로 전환된다."

## 3. 멤버 등급 체계

### 3.1 두 개의 등급

등급 구분은 단순하다. 검증된 핵심인재(Core)와, 핵심인재가 직접 소개하고 보증하는 인재(Intro)다. 직급도 연차도 기준이 아니다. 실력과 신뢰가 기준이다.

| 구분 | Core (핵심인재) | Intro (소개인재) |
|------|-----------------|-----------------|
| **정의** | 조직에서 누구나 인정하는 핵심인재 | Core가 직접 보증하는 차세대 핵심인재 |
| **진입 조건** | 기존 Core 추천 또는 밸류커넥트 직접 초대 + 확인 | Core 1명의 추천 필수 + 밸류커넥트 확인 |
| **연차 제한** | 없음 | 없음 (3년차도 가능) |
| **조찬 Meetup** | 우선 참석 | 빈자리 있을 시 참석 (Core 우선) |
| **멤버 디렉토리** | 전체 디렉토리 + 자기소개 작성 + 댓글/리액션 | 전체 디렉토리 + 자기소개 작성 + 댓글/리액션 |
| **커피챗 게시판** | 사연 작성 + 비밀 댓글 신청 모두 가능 | 비밀 댓글 신청만 가능 |
| **CEO 커피챗** | 세션 신청 가능 (우선 검토) | 세션 신청 가능 |
| **익명 게시판** | 읽기/쓰기 모두 가능 | 읽기/쓰기 모두 가능 |
| **채용 포지션** | EXCLUSIVE 48H 우선 접근 + 관심 표명 | AI 매치 포지션 열람 + 관심 표명 |
| **포지션 큐레이션 이메일** | 수신 (opt-in, AI 매칭 기반) | 수신 (opt-in, AI 매칭 기반) |
| **CEO 다이렉트 핫라인** | 제공 | 제공 |
| **셀프 소개 리워드** | 성사 시 500만원 | 성사 시 500만원 |
| **지인 추천 리워드** | 성사 시 수수료의 30% (약 600~1,500만원) | 해당 없음 |
| **독점 시장 데이터** | 월간 브리핑 제공 | 제공 안 됨 |
| **승급** | 최초 또는 Core 2명 추천 + 확인 | Core 2명 추가 추천 시 Core 승급 검토 |

### 3.2 핵심인재(Core)의 특성

다음 중 2개 이상 해당하는 사람이다.

- **조직 임팩트** — 이 사람이 나가면 팀이 흔들린다
- **동료 평판** — "한 명만 추천해줘" 하면 이름이 나온다
- **기술적 깊이** — "이 사람한테 물어봐" 수준
- **실행력** — 아이디어가 아니라 결과물로 증명한다
- **성장 궤도** — 5년 후 리더가 될 게 보인다
- **삶과 일의 일치** — 하는 일과 삶의 방향이 같다

### 3.3 핵심인재가 원하는 것

핵심인재의 가장 큰 외로움은 대화 상대가 없다는 것이다. 나와 같은 레벨에서, 이해관계 없이, 솔직하게 이야기할 수 있는 동료 — 이것이 돈 주고도 못 사는 것이다.

ValueConnect X가 제공하는 것:

- **Core 전용 디렉토리** — 검증된 폐쇄 네트워크에 소속되어 있다는 것 자체가 베네핏
- **사연 기반 커피챗 게시판** — 같은 급의 사람과 이해관계 없이, 맥락과 밀도가 일치하는 연결
- **CEO 커피챗 세션** — 의사결정자가 먼저 손을 내밀어 오는 경험
- **AI 매치 포지션** — 나에게 딱 맞는 기회만, EXCLUSIVE 48H 우선 접근
- **익명 게시판** — 이름 없이 꺼내는 진짜 고민, 평가 없이 집단 지성이 작동
- **독점 시장 데이터** — 외부에서 볼 수 없는 채용 데이터 및 업계 동향

### 3.4 Intro를 두는 이유

- Core가 직접 보증한 사람 → 퀄리티 자동 담보
- 3~5년 후 Intro가 Core로 성장 → 네트워크의 세대 교체
- Intro의 이직 성사 시 추천한 Core에게도 리워드 → Core의 추천 동기 강화

## 4. 비즈니스 모델

### 4.1 수수료 구조: 25%

일반 서치펌 수수료가 20~25%인 시장에서 ValueConnect X는 25%를 받는다. 비싼 게 아니다. 다른 곳에서는 절대 만날 수 없는, 검증된 핵심인재를 연결하기 때문이다.

| 케이스 | 구조 | 예시 (연봉 1억 기준) |
|--------|------|---------------------|
| **Case 1** — 셀프 소개 | 기업 25% → 이직자 500만원 고정 + 밸류커넥트 나머지 | 수수료 2,500만원 → 이직자 500만 / VC 2,000만 (80%) |
| **Case 2** — 지인 추천 | 기업 25% → 이직자 200만원 고정 + 추천인 수수료의 30% + 밸류커넥트 나머지 | 수수료 2,500만원 → 이직자 200만 / 추천인 750만 / VC 1,550만 (62%) |
| **Case 3** — CEO 커피챗 → 채용 성사 | CEO가 세션 공개 → 멤버 신청 → CEO 수락 → 채용 진행 → 성사 수수료 25% | 수수료 구조 동일. CEO 커피챗이 채용 퍼널의 입구가 됨 |

#### 4.1.1 Case 1 vs Case 2 보상 차등화 논리 (v4.2.1 신규)

| 구분 | Case 1 (셀프 소개) | Case 2 (지인 추천) |
|------|-------------------|-------------------|
| **이직자 역할** | 능동적 — 스스로 찾아와서 소개 | 수동적 — 누군가에 의해 발굴/설득됨 |
| **이직자 보상** | 500만원 (셀프소개 유인 보존) | 200만원 (이미 더 좋은 직장 + 연봉 인상이라는 본질적 보상 존재) |
| **추천인 역할** | 없음 | 후보자 발굴 + 관계 기반 신뢰 설득 + 이직 결심 유도 (= 헤드헌터 업무 대행) |
| **추천인 보상** | 없음 | 수수료의 30% |
| **설계 의도** | 셀프소개 유인 극대화 | 추천인의 헤드헌터 역할 기여를 정당하게 보상 |

> **보상 역전 방지:** 이직자 입장에서 "내가 직접 소개하면 500만원, 추천 받으면 200만원"이 명확히 구분되면, 셀프소개 유인이 오히려 강화된다.

### 4.2 리워드 구조 상세 (v4.2.1 변경)

| 리워드 유형 | 금액 | 조건 | 지급 시점 |
|------------|------|------|-----------|
| **셀프 소개 보상** | 500만원 고정 | 본인이 직접 소개 → 헤드헌팅 성사 | 성사 확인 후 14일 이내 |
| **지인 추천 이직자 보상** | 200만원 고정 | 추천을 통해 헤드헌팅 성사 | 성사 확인 후 14일 이내 |
| **지인 추천 수수료 (추천인)** | 수수료의 30% | 핵심인재 추천 → 헤드헌팅 성사 | 성사 확인 후 14일 이내 |

**지인 추천 리워드 시뮬레이션 (Case 2: 지인 추천 시나리오):**

| 이직자 연봉 | 수수료 (25%) | 이직자 보상 (200만원) | 추천인 리워드 (30%) | VC 순수익 | VC 마진율 |
|------------|-------------|---------------------|-------------------|-----------|-----------|
| 8,000만원 | 2,000만원 | 200만원 | 600만원 | 1,200만원 | 60% |
| 1억원 | 2,500만원 | 200만원 | 750만원 | 1,550만원 | 62% |
| 1.5억원 | 3,750만원 | 200만원 | 1,125만원 | 2,425만원 | 약 65% |
| 2억원 | 5,000만원 | 200만원 | 1,500만원 | 3,300만원 | 66% |

**셀프 소개 시뮬레이션 (Case 1: 비교 참고):**

| 이직자 연봉 | 수수료 (25%) | 이직자 보상 (500만원) | VC 순수익 | VC 마진율 |
|------------|-------------|---------------------|-----------|-----------|
| 8,000만원 | 2,000만원 | 500만원 | 1,500만원 | 75% |
| 1억원 | 2,500만원 | 500만원 | 2,000만원 | 80% |
| 1.5억원 | 3,750만원 | 500만원 | 3,250만원 | 약 87% |

> **v4.2.0 대비 변경 이유:** Case 2에서 이직자 보상(500만→200만)을 줄이고 추천인 비율(20%→30%)을 높였다. 추천인은 실질적으로 밸류커넥트가 직접 해야 할 후보자 발굴·설득·이직 결심 유도를 대행하는 역할이며, 이에 상응하는 보상이 필요하다. VC 순수익은 거의 유지되면서 추천인 리워드가 20~50% 상승한다.

#### 4.2.1 VC 마진 하한선 원칙 (v4.2.1 신규)

수수료의 **50% 이상**은 VC가 확보하는 구조를 하한으로 설정한다. 저연봉 케이스(8천만원 이하)에서도 운영 고정비용을 커버할 수 있어야 하기 때문이다.

|--------|---------------------------|-----------|

#### 4.2.2 추천인 티어 분리 (장기 검토, v4.2.1 신규)

## 5. 핵심 기능 상세

### 5.1 멤버 간 커피챗 — "사연을 올리고, 선택받는다"

PRD v4.1.3에서는 단순 포스팅 + 매칭 구조였으나, 디자인에서는 **사연(story) 기반 시스템**으로 확정되었다. 맥락과 밀도가 일치하는 연결을 만들기 위한 구조다.

#### 5.1.1 작동 방식

| 단계 | 내용 |
|------|------|
| **Step 1** — 사연 올리기 | 나의 상황, 고민, 찾고 있는 대화의 맥락을 솔직하게 작성 |
| **Step 2** — 비밀 댓글 신청 | 관심 있는 멤버가 비밀 댓글로 자신을 소개하며 커피챗 신청. 작성자만 볼 수 있음 |
| **Step 3** — 선택 후 연결 | 작성자가 댓글을 검토하고 대화를 원하는 사람을 직접 선택. 서로의 연락처가 공유됨 |

#### 5.1.2 게시판 구조

| 요소 | 내용 |
|------|------|
| **접근 권한 (사연 작성)** | Core 멤버만 사연 작성 가능 |
| **접근 권한 (비밀 신청)** | Core + Intro 모두 비밀 댓글로 신청 가능 |
| **필터** | 전체 / CORE / INTRO |
| **포스트 내용** | 멤버 등급 뱃지 + 이름·직책·회사 + 사연 제목 + 본문 + 해시태그 |
| **뱃지** | NEW (최근 게시), CORE/INTRO 등급 표시 |
| **비밀 신청 수** | 각 사연에 비밀 신청 건수 표시 (예: "비밀신청 12건") |
| **게시 일자** | 상대적 시간 표시 (예: "2일 전") |
| **유효 기간** | 포스팅 후 2주. 이후 자동 만료 (재게시 가능) |

#### 5.1.3 사연 예시

> **김민수 · VP of Product @Toss** [CORE] [NEW]
>
> **"AI 시대, PM의 역할에 대해 고민하시는 분"**
>
> 최근 AI 기술의 발전으로 PM의 역할이 크게 변하고 있습니다. 제품을 만드는 방식, 의사결정 과정, 팀과의 협업 방식이 모두 달라지고 있는데요...
>
> `#PM` `#AI` `#커리어전환`
>
> 2일 전 · 비밀신청 12건

#### 5.1.4 설계 철학

- **연결은 밀도가 만든다** — 약한 연결(weak ties)이 중요하지만, 그보다 더 중요한 것은 맥락과 밀도가 일치하는 연결
- **이 네트워크는 열려 있지 않다** — 그것이 이 네트워크의 가치
- **작성자가 연결을 통제한다** — 알고리즘이 아닌 사람이 선택하는 구조
- **비밀 신청의 의미** — 누가 신청했는지 다른 멤버는 알 수 없음. 부담 없이 신청 가능

### 5.2 CEO Coffee Chat — "의사결정자가 먼저 손을 내밀다"

기존 채용의 흐름은 후보자가 지원하는 것이다. ValueConnect X에서는 반대다. CEO가 먼저 핵심인재에게 커피챗 세션을 공개한다. 이것은 단순한 기능이 아니라, **핵심인재에 대한 예우의 시그널**이다.

#### 5.2.1 작동 방식

| 단계 | 내용 |
|------|------|
| **Step 1** — CEO가 직접 게시 | 조직의 맥락과 만나고 싶은 사람을 직접 작성 |
| **Step 2** — 멤버 비공개 신청 | ValueConnect X 멤버만 볼 수 있는 비밀 댓글로 지원 |
| **Step 3** — CEO 직접 선택 | CEO가 신청자를 검토하고 직접 수락 |
| **Step 4** — 맥락 있는 만남 | JD가 아닌 사람 대 사람으로 연결 |

#### 5.2.2 세션 구조

| 요소 | 내용 |
|------|------|
| **접근 권한 (세션 등록)** | CEO / C-level 의사결정자만 세션 등록 가능 |
| **접근 권한 (열람/신청)** | Core + Intro 멤버 모두 열람 및 신청 가능 |
| **세션 정보** | CEO 프로필 (이름/직책/회사/규모/업종) + 포맷 (온/오프라인) + 시간 (30분/45분/60분) + 일정 + 슬롯 수 |
| **슬롯 관리** | 세션별 총 슬롯 수 + 잔여 석 표시 (예: "2/5석 남음") + 마감 표시 |
| **대시보드 통계** | ACTIVE SESSIONS 수 / TOTAL SLOTS / REMAINING |
| **산업 필터** | 전체 / 오프라인 / 온라인 / 산업별 (Space Tech, Medical AI, B2B SaaS 등) |
| **신청 방식** | 비밀 댓글 — CEO만 신청자 정보 확인 가능 |
| **채용 연결 여부** | 커피챗 자체는 채용 목적 아님. 이후 채용 연결 시 표준 프로세스 적용 |

#### 5.2.3 CEO 세션 예시

> **김태현** · CEO · Krafton
> 게임 · 5,000명
> 오프라인 · 45분 · 2월 1주차
> joined 2일 전 · 2/5석 남음
>
> "배틀그라운드를 만들며 배운 것은, 글로벌 시장에서는 문화적 맥락이 제품의 성패를 결정한다는 것입니다..."

#### 5.2.4 전략적 가치

- **핵심인재 입장** — '내가 지원하는 게 아니라, CEO가 나를 찾아온다'는 경험
- **CEO 입장** — 공개 채용 없이, 검증된 핵심인재풀에 먼저 접근하는 채널
- **밸류커넥트 입장** — 자연스러운 채용 퍼널의 입구 — 커피챗 → 관심 → 채용 성사
- **네트워크 효과** — CEO들이 ValueConnect X에 상주할 이유가 생김
- **사회학적 가치** — 정보 비대칭(information asymmetry) 해소: 양쪽 모두 서류에 담길 수 없는 정보를 교환

### 5.3 익명 게시판 — "이름 없이 꺼내는 진짜 고민"

PRD v4.1.3에서 "익명 커리어 상담"으로 간략히 언급되었던 기능이 독립된 핵심 기능으로 확장되었다. 커리어의 고민, 연봉 협상의 불안, 조직 내 갈등 — 실명으로는 말할 수 없는 것들이 있다.

#### 5.3.1 게시판 구조

| 요소 | 내용 |
|------|------|
| **접근 권한** | Core + Intro 모두 읽기/쓰기 가능 |
| **익명 원칙** | 발신자는 익명, 신뢰는 실명 (멤버 인증 기반) |
| **카테고리** | 커리어 고민 / 연봉·협상 / 조직 갈등 / 이직 고민 / 번아웃 / 관계·리더십 |
| **게시물 정보** | 카테고리 + 게시 시간 + 조회수 + 제목 + 본문 미리보기 |
| **리액션 시스템** | 다양한 이모지 리액션 (공감, 인사이트, 응원 등) + 댓글 수 |
| **답글** | 모든 답글은 익명으로 처리 |
| **평가 금지** | 평가 없는 응답만 허용 |
| **CEO 접근** | CEO는 익명 게시판 열람 불가 (커뮤니티-채용 분리 원칙) |

#### 5.3.2 리액션 종류

| 이모지 | 의미 | 용도 |
|--------|------|------|
| 공감 | 나도 같은 고민이 있다 | 연대 표현 |
| 인사이트 | 좋은 관점이다 | 지적 공감 |
| 응원 | 힘내세요 | 정서적 지지 |
| 정확 | 핵심을 짚었다 | 내용 동의 |
| 방향 | 좋은 방향 제시다 | 답변 추천 |

#### 5.3.3 게시물 예시

> **연봉·협상** · 2시간 전 · 조회 342
>
> **"Series B 스타트업 CTO 오퍼 받았는데, 연봉 협상 어떻게 해야 할까요"**
>
> 현재 대기업 시니어 개발자로 재직 중이고, 스타트업 CTO 포지션 오퍼를 받았습니다. Base 연봉은 괜찮은데 스톡옵션 비중을...
>
> 공감 23 · 인사이트 14 · 댓글 2

#### 5.3.4 설계 철학

- **구조적 익명성** — Goffman의 인상 관리(impression management) 이론: 방어막을 낮춰 진짜 대화를 가능하게 하는 유일한 장치
- **멤버 인증 기반** — 아무나 쓸 수 없다. 익명이지만 검증된 사람들의 공간
- **집단 지성** — 같은 레벨의 경험을 가진 사람들의 응답은 다르다
- **평가 없음** — 답변은 도움을 주기 위해서만 존재한다

#### 5.4.3 포지션 큐레이션 이메일 (유지)

인앱 브라우징과 병행하여 이메일 채널도 유지한다.

| 단계 | 내용 |
|------|------|
| **Step 1** — 포지션 수집 | 기업 클라이언트(CEO)가 폐쇄 포지션 등록. 공개 JD가 아닌 비공개 포지션 위주. |
| **Step 2** — 멤버 프로필 분석 | 멤버의 LinkedIn 데이터, 전문 분야 태그, 경력 궤도, 현재 직급, 업계를 종합 분석 |
| **Step 3** — LLM 1차 매칭 | Claude Sonnet을 활용한 1차 적합도 스코어링 (100점 만점) |
| **Step 4** — LLM 2차 검증 | GPT-4o로 교차 검증. 1차와 2차 점수 편차가 클 경우 보류 |
| **Step 5** — LLM 3차 리스크 체크 | Claude Haiku로 멤버가 불쾌할 수 있는 포지션 필터링 |
| **Step 6** — 밸류커넥트 최종 확인 | AI 스코어 80점 이상 + 밸류커넥트 헤드헌터의 최종 판단 후에만 발송 결정 |
| **Step 7** — 개인화 이메일 발송 | 멤버 이름, 현재 역할, 왜 이 포지션을 제안하는지 이유를 담은 1:1 맞춤 메일 |

#### 5.4.4 이메일 설계 원칙

- **월 최대 2건** — 아무리 좋은 포지션이어도 한 명에게 월 2건 이상 발송하지 않는다
- **스코어 80점 이상만** — AI 검증 + 헤드헌터 확인을 통과한 것만 발송
- **발송 전 멤버 opt-in 확인** — 커리어 기회 이메일을 수신하겠다고 설정한 멤버에게만
- **이유를 설명한다** — '왜 이 포지션이 당신에게 맞는지'를 3문장 이내로 명시
- **단순 거절 가능** — 원클릭 '지금은 아니에요' 버튼. 이유 묻지 않음
- **피드백 루프** — 거절 패턴이 누적되면 매칭 로직 자동 개선

#### 5.4.5 LLM 검증 파이프라인

| 검증 단계 | 모델 | 검증 항목 | 기준 |
|-----------|------|-----------|------|
| 1차 적합도 분석 | Claude Sonnet | 경력 궤도 / 기술 스택 / 업계 / 연봉 밴드 정합성 | 80점 이상 통과 |
| 2차 교차 검증 | GPT-4o | 동일 기준으로 독립 평가 | 1차와 편차 15점 이내 |
| 3차 리스크 체크 | Claude Haiku | 멤버가 불쾌할 수 있는 포지션 필터링 (강등, 연봉 하락, 업계 이탈 등) | 이슈 없음 통과 |
| 최종 인간 검토 | 밸류커넥트 헤드헌터 | 전체 결과 종합 + 맥락 판단 | 발송 승인 |

### 5.5 멤버 디렉토리 — "같은 아비투스를 가진 사람들"

네트워크는 장(場, field)이다. 같은 아비투스(habitus)를 가진 사람들이 모였을 때, 이 공간은 사회적 자본이 실질적인 기회로 전환되는 공간이 된다.

#### 5.5.1 디렉토리 구조

| 요소 | 내용 |
|------|------|
| **검색** | 이름, 직군, 키워드로 검색 |
| **프라이버시** | "나와 ValueConnect X만 검색할 수 있습니다" |
| **자기소개 작성** | "+ 내 소개 작성" 버튼으로 프로필 등록 |
| **프로필 카드** | 이니셜 아바타 + 이름 + MEMBER 뱃지 + 직책/회사 + 전문 분야 태그 (최대 3개) + 가입일 |
| **소개 본문** | 자유 형식 자기소개 텍스트 |
| **전문 분야 (EXPERTISE)** | 상세 전문 분야 태그 목록 |
| **리액션** | 다양한 이모지 리액션 (공감, 인사이트 등) + 카운트 |
| **댓글** | 멤버 간 프로필에 댓글 작성 가능 |
| **접기/펼치기** | 프로필 카드 확장/축소 토글 |

#### 5.6.2 CTA 구조

- **Core 멤버 대상 (셀프 소개):** "핵심인재라면 직접 소개하세요" → 셀프 소개 → 헤드헌팅 성사 → 500만원
- **Core 멤버 대상 (지인 추천):** "좋은 사람을 알고 있다면 추천하세요" → 지인 추천 → 헤드헌팅 성사 → 수수료 30%

> **참고:** 지인 추천 리워드(수수료의 30%)는 Core 멤버 전용 혜택이다. Intro 멤버는 셀프 소개 보상(500만원)만 수령 가능하다. 멤버 혜택 페이지에서 Core/Intro 탭 전환 시 이 차이가 명확히 표시되어야 한다.
>
> **⚠️ 화면 검수 이슈:** 현재 디자인(/benefit)에서 CTA 섹션의 "추천하세요" 문구가 INTRO MEMBER 영역에 배치되어 있으나, 지인 추천은 Core 전용 혜택이므로 CORE MEMBER 영역으로 이동해야 한다. 또한 "수수료 20%"로 표기되어 있으므로 "수수료 30%"로 업데이트 필요.

### 8.1 원칙: 적게, 천천히, 확실하게

| 구분 | 조건 | 세부 |

### 8.2 퀄리티 유지 장치

- **추천인 책임** — 문제 멤버 발생 시 추천 Core에게도 알림
- **소수 유지** — Core 200명 + Intro 100명 = 총 300명 상한
- **비활성 확인** — 6개월 이상 조찬 불참 + 채널 비활성 시 정중한 확인
- **커뮤니티 가이드라인** — 자기 PR, 영업, 스팸 금지 → 경고 → 퇴장

## 9. 커뮤니티-채용 분리 원칙

이 원칙은 ValueConnect X 신뢰의 기반이다.

| 원칙 | 내용 |
|------|------|
| **CEO/기업은 멤버 영역 접근 불가** | 조찬, 사연 기반 커피챗, 익명 게시판 — 기업은 절대 접근 불가 |
| **커뮤니티 활동 ≠ 채용 데이터** | 멤버의 커뮤니티 활동 데이터는 채용 영역으로 넘어가지 않음 |
| **Opt-in 선택제** | 커리어 기회 수신을 원하는 멤버만 포지션 큐레이션 이메일 수신 및 익명 프로필 노출 |
| **CEO 커피챗은 세션 기반** | CEO가 세션을 공개하고, 멤버가 자발적으로 신청하는 구조. 강제 매칭 없음 |
| **멤버 커피챗 게시판은 멤버만** | CEO는 사연 기반 커피챗 게시판 열람 불가 |
| **익명 게시판은 멤버만** | CEO는 익명 게시판 열람 불가 |
| **익명화 레이어** | CEO 커피챗 신청 시 이름, 회사명, 연락처는 CEO 수락 후에만 공개 |
| **CEO only** | CEO(C-level)만 접근. 인사담당자는 밸류커넥트 인정 케이스 외 불가 |

## 11. 장기 비전: 20년 로드맵

| 시점 | 목표 |
|------|------|
| **Year 1** | Core 20명 → 100명 + Intro 30명. 관계 구축. 8개 페이지 플랫폼 런칭. 사연 기반 커피챗 + CEO 세션 + 익명 게시판 + AI 매치 포지션 가동. 첫 성사. |
| **Year 2~3** | Core 200명 + Intro 100명. 플랫폼 안정화. 연간 서밋. CEO 커피챗 → 채용 성사 사례 누적. |
| **Year 3~5** | Core들이 CTO, CEO, VP 레벨로 진입. CEO 핫라인의 양쪽이 모두 ValueConnect X 멤버인 상황 발생. |
| **Year 5~10** | 밸류커넥트 = '핵심인재를 가장 많이 아는 사람'으로 시장 포지셔닝. 리테이너 클라이언트 50곳+. |
| **Year 10~20** | 한국의 탑탤런트들을 글로벌 무대로 Sales. 1만 명 규모의 글로벌 핵심인재 네트워크. 아시아를 넘어 실리콘밸리, 유럽, 동남아까지. |

> **핵심 통찰:** 이 사업의 해자(moat)는 기술이 아니다. 코드는 복사할 수 있지만, 10년에 걸쳐 쌓은 Core 300명과의 신뢰는 복사할 수 없다. 그리고 사연 기반의 밀도 있는 연결, CEO가 먼저 손을 내밀어 오는 경험, 이름 없이도 진짜 고민을 나눌 수 있는 공간 — 이것이 ValueConnect X가 만드는 것이다.

## Appendix E: MCP Server & CLI Interface (v4.2.2 신규)

### E.1 개요

ValueConnect X를 MCP(Model Context Protocol) 서버로 제공하여, 엔지니어 후보자들이 Claude Code, Cursor 등 CLI 환경에서도 VCX의 핵심 기능에 접근할 수 있도록 한다. 엔지니어가 가장 익숙한 환경(터미널)에서 벗어나지 않고 커리어 네트워크를 활용할 수 있다.

**왜 MCP인가:**
- 핵심인재 중 엔지니어 비중이 높다 -- 터미널이 생활 공간
- Claude Code, Cursor 등 AI IDE에서 자연어로 VCX 기능 호출 가능
- 브라우저를 열지 않아도 포지션 탐색, 커피챗 작성, CEO 세션 신청 가능
- 개발 워크플로우에 자연스럽게 통합

### E.2 MCP 서버 도구 목록

| 도구 | 설명 | 접근 권한 |
|------|------|-----------|
| `vcx_dashboard` | 대시보드 - 매치 포지션, 알림, 커피챗 현황 | Core + Intro |

| `vcx_position_detail` | 포지션 상세 조회 | Core + Intro |
