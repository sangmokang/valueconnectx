## 1. 전체 구조 변경 사항

### 1.1 로그인 월(Login Wall) 적용

PRD 5.1 기준, VCX는 Invite-only Private Network입니다.
현재 사이트는 모든 페이지가 공개 접근 가능하나, **로그인 월 이후로 핵심 인터페이스를 이동**해야 합니다.

#### 비로그인 상태에서 접근 가능한 페이지

| 페이지 | 공개 범위 | 비고 |
|--------|----------|------|
| `/` (홈) | 전체 공개 | 현행 유지. 브랜딩 + 가입 유도 |
| `/service-overview` | 전체 공개 | 현행 유지. 5 Pillars 소개 |
| `/job-positions` | **목록만 공개** | 회사명 + 직무명 + AI Score만 노출. 상세(역할, 연봉 밴드, CEO 인용문, 관심 표명 버튼)는 블러 처리 + "로그인 후 확인" CTA |

#### 로그인 후에만 접근 가능한 페이지

| 페이지 | 접근 조건 |
|--------|----------|
| `/coffee-chat` | Member 인증 필수 |
| `/ceo-coffeechat` | Member 인증 필수 |
| `/anonymous-board` | Member 인증 필수 |
| `/member-directory` | Member 인증 필수 |
| `/job-positions` 상세 | Member 인증 필수 |

#### 디자인 요구사항

- **비로그인 상태로 `/coffee-chat`, `/ceo-coffeechat`, `/anonymous-board` 접근 시**: 로그인 유도 화면 표시
  - 중앙 정렬 레이아웃
  - "이 공간은 검증된 멤버만 접근할 수 있습니다"
  - [로그인] / [멤버십 신청하기 →] 버튼
  - 배경: 해당 페이지의 블러 처리된 콘텐츠 (엿보기 효과)

- **네비게이션 변경**: 비로그인 시 커피챗/CEO Coffeechat/익명 게시판 메뉴에 🔒 아이콘 추가

## 2. 페이지별 수정 사항

### 2.1 홈페이지 (`/`)

#### 히어로 섹션 — 변경 없음
현행 유지. "소수의 핵심인재들의 작은 네트워크, ValueConnectX"

#### Features 섹션 — 항목 추가

현재 6개 Feature:
1. Core 전용 디렉토리
2. LLM 포지션 큐레이션
3. CEO Coffee Chat
4. 멤버 간 커피챗 게시판
5. 익명 게시판
6. 폐쇄 채용시장 인사이트

**추가 또는 수정:**
- 항목 5 "익명 게시판" 설명에 **"이 회사 어때요?"** 카테고리 언급 추가
  - 변경 전: "이름 없이 솔직하게. 조직, 커리어에 대한 진짜 이야기를 꺼낼 수 있는 공간."
  - 변경 후: "이름 없이 솔직하게. 조직, 커리어, 그리고 '이 회사 어때요?' — 진짜 이야기를 꺼낼 수 있는 공간."

#### 멤버십 섹션 — 용어 통일

**현재 사이트**: CORE / INTRO
**PRD 5.1**: Core / Endorsed

> **결정 필요**: PRD에서는 "Endorsed"이나 사이트에서는 "Intro"를 사용 중. Branding.md에서도 "Intro"를 사용. **"Intro"로 통일하고 PRD를 수정하는 것을 권장** (사용자 친화적 용어).

#### 연결 섹션 — Hiring Risk Filter 메시지 추가

현재 3개 카드: CEO Coffeechat / 멤버 간 커피챗 / 익명 게시판

**섹션 하단에 신뢰 배지(Trust Badge) 추가:**
```
◈ 경영자 검증 완료     ◈ 포지션 품질 관리     ◈ 멤버 정보 보호
  CEO/Founder 직접 참여    Admin 내부 등록만      Anti-Scraping 적용
```

이는 PRD 6. Hiring Risk Filter의 핵심 메시지를 시각적으로 전달합니다.

### 2.3 익명 게시판 (`/anonymous-board`)

**현재 (사이트)**
```
[전체] [커리어 고민] [연봉·협상] [조직 갈등] [이직 고민] [번아웃] [관계·리더십]
```

**변경 (PRD 5.1)**
```
[전체] [커리어 고민] [연봉·협상] [조직 고민·리더십] [번아웃] [생산성·News] [이 회사 어때요?]
```

변경 상세:
- "조직 갈등" → "조직 고민·리더십" (범위 확대)
- "이직 고민" → 삭제 (커리어 고민에 포함)
- "관계·리더십" → "조직 고민·리더십"에 통합
- **"생산성·News" 추가** (신규)
- **"이 회사 어때요?" 추가** (신규, PRD 5.6 반영)

#### "이 회사 어때요?" 카테고리 UI

이 카테고리는 특별한 운영 정책이 적용되므로 시각적으로 구분합니다.

**카테고리 선택 시 상단 안내 배너:**
```
┌──────────────────────────────────────────────┐
│ ℹ️  이 카테고리는 사실 기반 정보만 허용됩니다.     │
│     감정적 비방 및 근거 없는 주장은 삭제됩니다.     │
│     모든 글은 채용 활용이 불가합니다.               │
└──────────────────────────────────────────────┘
```

**글 작성 시 동의 체크박스:**
- [ ] 작성 가이드라인을 확인했습니다 (사실 기반, 비방 금지)
- [ ] 이 글은 채용 목적으로 활용되지 않음을 이해합니다

**게시글 카드 — 기존 형식 유지하되 카테고리 태그 색상 차별화:**
- 일반 카테고리: Slate (#64748B) 배경
- "이 회사 어때요?": Gold (#EAB308) 배경 + 약간 어두운 텍스트
